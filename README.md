# chess
chesh in dash, yesh

## how to run

* install stockfish. on linux:
```
sudo apt install stockfish
```

* install anaconda
* install conda environment

```
conda env create -f conda.yml
```

* activate environment

```
conda activate chess
```

* run app (from project folder)

```
python -m app
```
